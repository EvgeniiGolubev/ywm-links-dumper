# SEO Tools

SEO Tools – это мощная и легкая утилита для SEO-специалистов, объединяющая в себе автоматизацию Яндекс.Вебмастера и проверку индексации страниц через эмуляцию реального браузера.

## Что нового?
- Теперь настройки и данные хранятся рядом с программой.
- В комплекте идет оптимизированная версия браузера для проверки индексации.
- Использует Playwright для обхода защит. Если появляется капча – вы решаете её вручную, и бот продолжает работу.

## Основные функции

1. Выгрузка ссылок (Yandex Webmaster)
   - Автоматический сбор всех внешних ссылок через официальный API. 
   - Сохранение в XLSX с автоматическим именованием. 
   - Smart Highlighting: автоматическая подсветка «мусорных» доменов (казино, порно, фарма) красным цветом для быстрой чистки ссылочного профиля.
2. Проверка индексации
   - Массовая проверка страниц в поисковых системах. 
   - Использование реального профиля браузера для сохранения сессий. 
   - Работа в режиме headless (скрытно) с возможностью вмешательства пользователя для решения капчи.

## Инструкция по сборке проекта

### Подготовка окружения
```bash
  python -m venv .venv
  .venv\Scripts\activate
  pip install -r requirements.txt
```
### Подготовка браузера (Chromium)
Для работы проверки индексации необходимо установить Chromium локально в проект:
```bash
  $env:PLAYWRIGHT_BROWSERS_PATH="pw-browsers"
  playwright install chromium
```

### Сборка проекта
```bash
  pyinstaller --noconsole --onefile --name "SEO Tools" --icon=assets/icon.ico --add-data "assets;assets" app.py
```

## Получение доступа к API

1. Зарегистрируйте приложение в [Яндекс.OAuth](https://oauth.yandex.ru/)
2. В доступах укажите `webmaster:hostinfo`
3. Сформируйте ссылку: `https://oauth.yandex.ru/authorize?response_type=token&client_id=MY_CLIENT_ID&scope=webmaster:hostinfo`
4. Перейдите по ней, разрешите доступ, скопируйте токен.

## Ограничения
- API Яндекс.Вебмастера не отдает анкорные тексты ссылок. 
- Проверка индексации зависит от стабильности прокси (если используются) и частоты появления капчи.
